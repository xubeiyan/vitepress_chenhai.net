## Tmux

> [Tmux](https://github.com/tmux/tmux/wiki) 是 Linux 的一款终端复用器（Terminal Multiplier），简单来说就是可以在一个终端窗口下打开多个终端，那些限制你未注册的时候打开有限个Tab的终端软件（例如XShell）有什么用嘛
